# 🤝 Contributing

Thank you for your interest in contributing to **Blender Pancake Scene**!

---

## 📋 Workflow

1. Check existing [Issues](../../issues) before creating a new one
2. Create an Issue and set a Priority label
3. Work on your changes
4. Submit a Pull Request and link the related Issue

---

## 🐛 Bug Reports

- Describe what went wrong clearly
- Include steps to reproduce the issue
- Mention your Blender version and OS

---

## ✨ Feature Requests

- Explain your idea clearly
- Describe why it would improve the scene

---

## 🎨 Contributions Welcome

- Visual improvements to the scene
- Material or lighting enhancements
- New props or scene variations
- README or documentation improvements

---

## 🔧 Pull Requests

- Keep changes focused and minimal
- Always link a related Issue
- Test your changes in Blender before submitting

---

## 🎯 Rules

- Be respectful and constructive
- Keep things simple and organized
- Follow the [Code of Conduct](CODE_OF_CONDUCT.md)
