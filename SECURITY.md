# 🛡️ Security Policy

## 📦 Supported Versions

| Version | Supported |
|---------|-----------|
| 3.4  | ✅ Yes     |

---

## 📢 Reporting a Vulnerability

If you discover a potential security issue in this repository, please report it privately.

**Do not open a public Issue for security-related problems.**

You can report privately via:
👉 [GitHub Private Vulnerability Reporting](../../security/advisories/new)

---

## 🔍 Scope

This repository contains Blender (`.blend`) files and related assets only.

Typical concerns include:
- Malicious scripts embedded in `.blend` files
- Unexpected or dangerous Python code within the scene

---

## ⚡ Safety Notes

- Always open `.blend` files using an **official release of Blender**
- Use a **trusted environment** when testing files from unknown sources
- Review **scene scripts and Python drivers** before running them
- Disable **Auto Run Python Scripts** in Blender preferences if unsure

---

## ⏱️ Response Time

We will review and respond to reports as soon as possible.

Thank you for helping keep this project safe! 🙏
